# Claude Code Context Visualization

This project is a visualization tool for the context provided to Claude in a coding-related conversation.

[**View the live deployment on GitHub Pages**](https://noindexing.github.io/claude-context-visualization)

## About

This application helps to visualize the context that is being sent to the Claude AI model during a conversation. It's built with React, TypeScript, and Vite, and deployed using GitHub Pages.

## Development

To run this project locally, follow these steps:

1.  Clone the repository:
    ```bash
    git clone https://github.com/noindexing/claude-context-visualization.git
    ```
2.  Install the dependencies:
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm run dev
    ```

## Technologies Used
